# Genio + GAAD 2026
This repo contains a challenge that will sent out to celebrate GAAD 2026 at Genio. The challenge is a 5 page website where participants need to use assistive technology to gather clues to a password.

The pacticipants that complete it will be able to submit their name and answer to a Google Form which will deposit the details in a Google Sheet.

This was all vibe coded so... yeah...
