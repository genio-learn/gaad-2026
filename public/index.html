Welcome to the Lab.
Things work differently here.

This is a reverse accessibility lab. Each room is intentionally broken for mainstream use — but works perfectly if you know the right way in.

No prior experience with assistive technology needed. Each room gives you a hint. The goal is to feel the friction first — then discover the "other way" that makes everything click.

How it works

  1. 1
    Enter each challenge room There are 5 rooms, each focused on a different way of interacting with a page.
  2. 2
    Find the hidden clue Each room hides a one-word clue. The mainstream path won't reveal it — you'll need to interact differently.
  3. 3
    Collect all five clues Jot them down. They'll make sense when you reach the final quiz.
  4. 4
    Answer the final quiz Combine your clues to unlock the answer and finish the challenge.

The five rooms

  • ⌨️ Room 1 — Keyboard only

    Mouse? Useless. Find the clue using nothing but your keyboard.

    Keyboard navigation
  • 🔊 Room 2 — Screen reader

    The visual layout is a dead end. The semantic layer tells a different story.

    Screen reader
  • 🎙️ Room 3 — Voice control

    Mouse is dead. Keyboard is a maze. One button has the name you need to say.

    Voice control
  • 🔎 Room 4 — Zoom & magnification

    The control you need is there. The layout is hiding it. Zoom in and watch everything shift.

    Zoom & reflow
  • 🌀 Room 5 — Reduced motion

    The page is alive with interference. Calm the signal to extract the hidden code.

    Motion & vestibular
